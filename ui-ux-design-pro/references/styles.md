# Styles

Use styles as a direction-setting tool, not a decoration layer.

## Catalog
1. **Glassmorphism** — Layered translucent panels for premium dashboards; use with strong contrast and limited blur.
2. **Neumorphism** — Soft raised surfaces for lightweight utility apps; avoid for dense data tables.
3. **Brutalism** — Hard edges and loud hierarchy for bold brand statements; use sparingly in core workflows.
4. **Bento Grid** — Modular card composition for AI, productivity, and showcase dashboards.
5. **Dark Mode Native** — Design first in dark surfaces with strong semantic contrast and restrained saturation.
6. **Minimal Swiss** — Grid-led, neutral, typography-forward UI for enterprise trust.
7. **Editorial Tech** — Magazine-like hierarchy for landing pages, reports, and thought leadership products.
8. **Cyber SaaS** — Electric accents on dark canvases for devtools and security products.
9. **Warm Fintech** — Muted neutrals with one trust color for financial clarity.
10. **Playful Productivity** — Rounded modules and cheerful accents for collaborative tools.
11. **Monochrome Pro** — Single-hue dominance for calm, premium interfaces.
12. **Soft Gradient** — Ambient gradient backgrounds for onboarding and hero areas.
13. **High Contrast Utility** — Accessibility-led strong contrast for operational tools.
14. **AI Assistant Shell** — Chat-first layout with pinned context and action rails.
15. **Data Dense Control Room** — Compact spacing, sticky scaffolds, status-first patterns.
16. **Command Palette First** — Keyboard-centric UI with layered quick actions.
17. **Card Analytics** — Reusable summary cards with trend and alert slots.
18. **Marketplace Modular** — Browsing-first layouts with strong filter/sort scaffolds.

22. **Industrial Ops** — Rugged utility styling with alert visibility prioritized.

## Style selection heuristic
- Choose lower-noise styles for operational and enterprise-heavy products.
- Choose expressive styles for acquisition surfaces, launches, or brand-led experiences.
- Separate marketing style from app style when clarity would otherwise drop.
- Test dark mode, density, and accessibility before locking a style.
