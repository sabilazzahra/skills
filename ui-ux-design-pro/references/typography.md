## Rules
- Default to 16px body for desktop product UI unless density demands 14px.
- Keep heading scale tighter in dashboards than in marketing surfaces.
- Use monospace only for code, IDs, metrics, or structured input where scan accuracy matters.
- Validate numeral clarity for analytics and finance products.
