## Context rules
- Reserve the brightest hue for action, not decoration.
- Keep status colors independent from brand primary when operational accuracy matters.
- For dark mode, reduce saturation on large fills and raise contrast on text/borders.
- In data viz, avoid using the same hue family for both status semantics and category encoding.
